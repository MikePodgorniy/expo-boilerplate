## Expo boilerplate

[![Build Status](https://travis-ci.com/ebkn/expo-boilerplate.svg?branch=master)](https://travis-ci.com/ebkn/expo-boilerplate)

ReactNative(Expo) + TypeScript + Storybook + Jest boilerplate

### Dependencies
```
Node.js v10.10.0
yarn
Expo v31.0.6
React v16.5.0
ReactNative v31.0.0
```

### Usage
```
# run expo server
$ yarn start

# run storybook server
$ yarn storybook

# linter
$ yarn lint
$ yarn lint:fix

# test
$ yarn test
```
